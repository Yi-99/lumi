Keys are encrypted before they touch disk.

How your keys are protected: in flight, every request uses HTTPS/TLS and keys are sent only to the provider they belong to. At rest, keys are encrypted with AES-GCM-256 before storage; the master key is a non-extractable browser key whose raw bytes are never exposed to JavaScript. Keys stay on this device (never synced in plaintext). If a local proxy server is configured, keys are forwarded per-request in headers and relayed to the providers — the server never stores or logs them. No browser extension can protect keys from malware running on your OS.
